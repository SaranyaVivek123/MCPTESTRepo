# Hello World Java Program

This is a simple Hello World program written in Java to demonstrate basic Java program structure.

## Prerequisites
- Java Development Kit (JDK) installed on your system
- Basic understanding of command line operations

## How to Compile and Run

1. Open a terminal/command prompt
2. Navigate to the `src` directory
3. Compile the program:
   ```bash
   javac HelloWorld.java
   ```
4. Run the program:
   ```bash
   java HelloWorld
   ```

## Expected Output
```
Hello, World!
```

## Program Structure
- The program consists of a single class named `HelloWorld`
- The `main` method is the entry point of the program
- It uses `System.out.println()` to display text to the console

## Notes
- Make sure your Java environment variables are properly set
- The file name must match the class name (`HelloWorld.java`)
